# Andes.IA Backend

Initial FastAPI scaffold for the Andes.IA backend.

## Structure
- `app/api` — HTTP routes and API layer
- `app/agents` — coaching/automation agent implementations
- `app/database` — database/session/integration helpers

## Local development
1. Create a virtual environment
2. Install dependencies from `requirements.txt`
3. Run the app with `uvicorn app.main:app --reload`
